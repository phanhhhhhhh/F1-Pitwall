### Key Learnings
1. Spring Boot 4.0.5 / Spring Framework 7:
   - TestRestTemplate package (org.springframework.boot.test.web.client) doesn't exist
   - DefaultResponseErrorHandler.handleError has different signature
   - Workaround: catch HttpClientErrorException for 4xx assertions
2. Bucket4j rate limit on /api/auth/login (5 req/min) requires PER_CLASS test lifecycle
3. OpenF1 rate limit (3 req/s + 30 req/min) requires staggering parallel API calls (400ms)
4. JUnit 5 creates new test instance per method by default — @TestInstance(PER_CLASS) needed for shared auth tokens
